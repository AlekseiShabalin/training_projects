Tracker - программа создания, редактирования, хранения заявок


ru.ashabalin - пакет содержащий файлы программы
Tracker - класс содержащий методы добавления, редактирования, удаления, получения списка заявок, поиска по Id и по имени.
Item - класс описывающий заявку. Содержит имя, описание, ID, время создания заявки.
Task -  класс унаследованный от Item, с его помощью ведется работа по созданию заявок.
StartUI - класс запускающий программу.
